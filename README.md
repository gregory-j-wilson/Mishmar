# Mishmar (מִשְׁמָר)

A Rule of Life builder for Christian and Messianic Jewish spiritual practices.

## Features
- Track daily, weekly, monthly, and seasonal spiritual practices
- AI-powered practice suggestions
- Persistent local storage
- Beautiful dark theme with pastel accents

## Tech Stack
- React 18
- Tailwind CSS
- Anthropic Claude API
- Persistent Storage API
